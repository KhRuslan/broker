# ТОО «ЖАНА ТАЛАП БРОКЕР»

Корпоративный сайт таможенно-брокерской компании.

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Структура

- **Главная** — hero, о компании, преимущества, услуги, география, почему нам доверяют, контакты, форма
- **Услуги** — импорт, экспорт, транзит, разрешительная документация
- **Контакты** — контактные данные, форма заявки, карта
